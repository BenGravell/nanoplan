# CommonRoad scenario corpus

The checked-in XML files are original to this repo, generated by
[`tools/generate_diverse_scenarios.py`](../../tools/generate_diverse_scenarios.py),
and covered by the repo's MIT [license](../../LICENSE). Each is a CommonRoad
2020a lanelet network, obstacle set, and planning problem.

Official CommonRoad scenario payloads are deliberately **not** checked in.
[`tools/load_commonroad_scenarios.py`](../../tools/load_commonroad_scenarios.py)
fetches a pinned TUM-authored subset and caches both its XML and converted
nanoplan JSON outside the worktree:

```sh
cargo run -- "$(uv run tools/load_commonroad_scenarios.py)"
```

The loader uses `$NANOPLAN_CACHE_DIR` when set, otherwise the platform cache
directory (`$XDG_CACHE_HOME/nanoplan/commonroad` on Linux). Its stdout is only
the converted directory; Rich status and cache-hit output go to stderr so the
command composes with the viewer and batch tools.

## How the checked-in corpus is used

[`tools/export_commonroad_scenarios.py`](../../tools/export_commonroad_scenarios.py)
converts CommonRoad XML into the JSON the viewer and batch runner load:

```sh
python3 tools/export_commonroad_scenarios.py scenarios/commonroad scenarios/web
python3 tools/bundle_web_scenarios.py
cargo run --release --bin batch -- --count 0 --dir scenarios/web
```

Two conversions are checked in at [`scenarios/json/`](../json/) and compiled
into the viewer binary (`ZAM_BrakingLead-1_1_T-1`, `ZAM_CutIn-1_1_T-1`);
the rest ship to the web build via `scenarios/web_bundle.json`.

CommonRoad XML compatibility is not redistribution permission. The cached
loader limits itself to TUM-affiliated scenarios marked `hand-crafted` or
`Specification-based scenario synthesis`; keep other downloaded datasets
local unless their applicable license explicitly permits redistribution.

## Regenerating the local corpus

```sh
python3 tools/generate_diverse_scenarios.py --clean
```

Deterministic per seed — the default settings reproduce this directory
byte-for-byte. `--variations N` generates N randomized variations per
scenario category instead of 1.
